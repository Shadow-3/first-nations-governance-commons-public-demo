Review Pack

Use this hub to review the fake-data public demo, advisor materials, governance drafts, and Decidim sandbox build handoff. Real community data, member lists, cultural material, and real decision records are deliberately excluded.

Package Index
Advisors Review the concept, score readiness, and record go/no-go conditions. Funders Check the decision case, budget options, risk controls, and partner fit. Builders Use the Builder Room, blueprint, component map, acceptance tests, and handover plan. Contributors Review the open-source model, maintainer rules, roadmap, and issue templates. Sprint Team Open the first fake-data Decidim sprint board, issue pack, RACI, and evidence log. Ops Team Run admin configuration, QA scenarios, evidence, risk, handover, and go/no-go controls. Delivery Team Run the live GitHub milestone, blockers, status labels, issue links, and sprint updates. Assurance Team Check evidence gates, sign-off roles, no-go triggers, and fake-data boundary. Launch Team Choose the sandbox build path, check environment readiness, and map Decidim outputs. Seed Team Prepare Decidim-shaped fake spaces, components, content records, and safety holds. Build Pack Use environment checks, runbook, provider handoff, and acceptance tests. Partner Desk Send the RFQ pack, compare providers, score responses, and apply no-go gates. Delivery Control Control provider milestones, evidence, payment gates, RACI, and handover. Organisation Onboarding Test organisation and initiative join paths with the Toastmasters First Nations controlled sample. Pilot Readiness Check private-pilot gates, test scenarios, role access, data transfer, and closeout. Pilot Ops Run daily controls, participant states, incidents, official handoff, and closeout evidence. Pilot Build Prepare private environment, minimum data schema, access register, dry run, and go/no-go decision. Pilot Launch Open the controlled private pilot, run first-30-day learning, and decide proceed, hold, or no-go. Pilot Closeout Close the controlled pilot, prove deletion/retention, and decide proceed, hold, no-go, archive, or second pilot. Authority Approval Approve or hold real pilot activation through authority, data governance, privacy, role access, notice, and scale gates. Activation Drill Rehearse signoff, tabletop scenarios, evidence packet, decision record, and first-72-hour monitoring. Control Tower Review stage readiness, blockers, evidence, owners, and decision routes in one place. Evidence Vault Prepare Dean advisory review, private evidence vault, access rules, and authority meeting gates. Action Docket Convert advisory findings into owner actions, authority motions, and readiness checks. Decidim Blueprint Spaces, components, permissions, seed records, hosting controls, and acceptance tests. Governance/Data Review authority, privacy, data sovereignty, classification, and security guardrails.
