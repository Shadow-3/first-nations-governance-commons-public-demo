Review Pack

Use this hub to review the fake-data public demo, advisor materials, governance drafts, and Decidim sandbox build handoff. Real community data, member lists, cultural material, and real decision records are deliberately excluded.

Package Index
Advisors Review the concept, score readiness, and record go/no-go conditions. Funders Check the decision case, budget options, risk controls, and partner fit. Builders Use the Builder Room, blueprint, component map, acceptance tests, and handover plan. Contributors Review the open-source model, maintainer rules, roadmap, and issue templates. Sprint Team Open the first fake-data Decidim sprint board, issue pack, RACI, and evidence log. Ops Team Run admin configuration, QA scenarios, evidence, risk, handover, and go/no-go controls. Pilot Readiness Check private-pilot gates, test scenarios, role access, data transfer, and closeout. Pilot Ops Run daily controls, participant states, incidents, official handoff, and closeout evidence. Pilot Build Prepare private environment, minimum data schema, access register, dry run, and go/no-go decision. Pilot Launch Open the controlled private pilot, run first-30-day learning, and decide proceed, hold, or no-go. Pilot Closeout Close the controlled pilot, prove deletion/retention, and decide proceed, hold, no-go, archive, or second pilot. Authority Approval Approve or hold real pilot activation through authority, data governance, privacy, role access, notice, and scale gates. Action Docket Convert advisory findings into owner actions, authority motions, and readiness checks. Decidim Blueprint Spaces, components, permissions, seed records, hosting controls, and acceptance tests. Two-Track Pilot Barayamal organisation onboarding plus FN Toastmasters member tester. Test Lab Fake accounts, scripts, observer notes, issue triage, gates, and evidence. Scorecard Pass, hold, blocked, remediations, rollout, and public summary. Closeout Private fake-data run board, evidence gates, approvals, public update, and launch choice. Gate Private approval owners, motions, gates, risks, tickets, sandbox checks, and decisions. Docket Meeting agenda, evidence custody, owner decisions, redactions, outcomes, actions, audit, and comms. Ledger Release locks, carry-forward mapping, access controls, acceptance evidence, rollback, and comms. Pilot Runbook Barayamal, Toastmasters, Dean review, and gated Decidim build execution. Control Room Rehearsal schedule, sign-offs, issues, scorecard, and handoff actions. Decision Handoff Owner votes, blocker closure, build route, tickets, and public-safe communications. Owner Meeting Ballots, quorum, blocker closure, RFQ, tickets, minutes, public lines, and actions. Execution Control Evidence gates, owner actions, build activation, rollback, public lines, and next review. Session Runner Run the closeout meeting rehearsal. Handoff Prepare private owner handoff packets.
