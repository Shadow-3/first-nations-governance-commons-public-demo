Review Pack

Use this hub to review the fake-data public demo, advisor materials, governance drafts, and Decidim sandbox build handoff. Real community data, member lists, cultural material, and real decision records are deliberately excluded.

Package Index
Advisors Review the concept, score readiness, and record go/no-go conditions. Funders Check the decision case, budget options, risk controls, and partner fit. Builders Use the Builder Room, blueprint, component map, acceptance tests, and handover plan. Contributors Review the open-source model, maintainer rules, roadmap, and issue templates. Sprint Team Open the first fake-data Decidim sprint board, issue pack, RACI, and evidence log. Ops Team Run admin configuration, QA scenarios, evidence, risk, handover, and go/no-go controls. Governance/Data Review authority, privacy, data sovereignty, classification, and security guardrails.

Package rule

Only fake/sample data belongs in this public demo. A real pilot requires separate community governance approval, privacy review, data sovereignty controls, cultural safety review, hosting approval, and a clear support process.
