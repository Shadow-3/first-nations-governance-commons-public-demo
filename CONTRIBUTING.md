# Contributing To The First Nations Governance Commons Public Demo

Initial developer/custodian: Barayamal / Dean Foley  
Mode: public demo and fake-data sandbox support only

## Core Rule

Open-source the infrastructure. Do not open-source community authority.

This repository can accept contributions to public demo code, documentation, fake/sample data, Decidim configuration guidance, accessibility, testing, and implementation planning.

It must not accept:

- Real community records.
- Member lists.
- Cultural material.
- Real voting or eligibility data.
- Sensitive personal information.
- Claims that a real community decision has been made.

## Contributor Lanes

- Public demo improvements.
- Fake-data Decidim sandbox configuration.
- Documentation and plain-language review.
- Accessibility and mobile usability.
- Security, privacy, and data-sovereignty review.
- Testing and acceptance checks.
- Funding, partner, and implementation planning.

## Before Opening An Issue

Check whether the request belongs in this public repository.

Use this repository for infrastructure and fake-data sandbox work.

Do not use this repository to make or challenge real community decisions.

## Before Opening A Pull Request

Every pull request should confirm:

- The change uses fake/sample data only.
- No cultural material or private records were added.
- The change does not alter real community authority.
- Any new public page keeps the fake-data boundary visible.
- Mobile layout was checked at phone width when UI changed.
- Relevant docs, data, manifest, and package index files were updated.

## Maintainer Model

Barayamal / Dean Foley is the initial developer/custodian.

Additional maintainers should be added only after trust, role clarity, and governance boundaries are agreed.

Community authority remains separate from code maintainership.
