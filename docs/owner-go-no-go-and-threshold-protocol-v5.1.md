# Owner Go/No-Go and Threshold Protocol v5.1

## P0 Rule

Every P0 stress scenario needs private owner acceptance before any live pilot or provider access to real records can be considered.

## Threshold Outcomes

- **Pass**: owner accepts private evidence and rollback proof.
- **Hold**: owner requests more evidence or a repeat drill.
- **No-go/archive**: repeated P0 failure, unclear authority, privacy gap, official-boundary risk, support gap, or evidence custody failure.

## Barayamal and First Nations Toastmasters Boundary

Barayamal remains the controlled organisation sample. First Nations Toastmasters remains a guest/member-boundary sample. Neither path authorises real founder intake, business-sensitive records, official Toastmasters records, member data, payments, education records, support cases, or official decisions in Decidim.
