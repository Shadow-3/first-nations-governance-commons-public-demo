# Owner Evidence Acceptance Protocol v5.0

Use this to decide whether an evidence gate can move from blocked or hold to accepted.

## Accept

Accept evidence only when the owner, evidence location, decision scope, privacy/IDS/data boundary, rollback path, and public replacement line are clear.

## Reject

Reject evidence if it relies on advisory approval, public-demo content, private records in public files, unclear authority, real participant data, official Toastmasters records, credentials, or logs.

## Custody

Private evidence stays outside the public static package. Public artifacts can show aggregate status, public-safe lines, and fake/sample records only.

## No Substitution

Dean Foley / Barayamal advisory input can improve the work, but it cannot replace owner votes or accepted evidence gates.
