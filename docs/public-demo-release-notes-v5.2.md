# Public Demo Release Notes v5.2

v5.2 adds the Private Stress Lab Run Pack.

The v5.1 package defined the stress tests. v5.2 turns them into an operational fake/sample run kit for the first private session:

- 12 session steps.
- 11 P0 stress runs.
- 12 owner roles.
- 16 private evidence log items.
- 28 controlled Barayamal and First Nations Toastmasters sample records.
- 10 provider boundary tests.
- 10 approved communications lines.
- 12 go/no-go decision records.
- 111 total run readiness records.

This release does not approve live intake, provider access to real records, outbound email, official Toastmasters workflows, or live Decidim access.
