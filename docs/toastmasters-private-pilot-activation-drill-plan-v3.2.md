# Toastmasters First Nations Activation Drill Plan v3.2

Mode: controlled public sample for planning only.

## Purpose

Use this drill to test whether First Nations Toastmasters can safely move from authority approval into tightly controlled private intake.

## Required roles

- Pilot authority
- Governance reviewer
- Data reviewer
- Privacy reviewer
- Safety reviewer
- Technical custodian
- Toastmasters launch lead
- QA reviewer
- Access support contact

## Controlled sample pathway

1. Use fake participant records only.
2. Confirm official Toastmasters membership, chartering, payment, signatures, and member numbers remain outside the public package and outside any unapproved pilot tool.
3. Run all tabletop scenarios before real intake.
4. Record the decision privately and publish only a redacted status.
5. Recheck at hour 0, 2, 4, 8, 12, 24, 36, 48, and 72 after any approved activation.

## Hold rule

Hold the pilot if any required signoff, support route, access test, backup/delete drill, privacy question, official boundary, or public redaction QA is incomplete.
