## Decision route options

- ROUTE-001: Use public demo only (Ready) - Public review can continue with fake/sample data only
- ROUTE-002: Hold for authority approval (Hold) - Authority, IDS, privacy, or official boundary not approved
- ROUTE-003: Hold for activation drill (Hold) - Authority model exists but signoff/tabletop/evidence/monitoring not rehearsed
- ROUTE-004: Prepare private environment (Hold) - Technical proof is missing or stale
- ROUTE-005: Provider quote with fake data (Ready) - Provider help needed but no real data access approved
- ROUTE-006: No-go real activation (Blocked) - Any blocked no-go trigger is present
- ROUTE-007: Conditional activation (Hold) - All blocked items closed but support/provider/advice/closeout conditions remain
- ROUTE-008: Approve limited private pilot (Ready) - All required private evidence, signoffs, drill tasks, and monitoring checks pass

## Public boundary

This brief can name status, roles, and generic blockers. It must not include real names, contact details, incidents, access needs, official records, payments, member numbers, signatures, access logs, private decisions, or private evidence.
