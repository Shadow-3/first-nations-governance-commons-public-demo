# Session Export and Public Status Guide v5.6

Allowed exports from the public demo:

- Private minute shell with fake/sample placeholders.
- Owner action queue.
- Provider refusal extract.
- Public-safe status update.
- Repeat-run hold notice.
- No-go/archive note.
- Sample revision queue.
- Advisory readout.

Excluded from public exports:

- Real owner signatures.
- Real participant records.
- Official Toastmasters records.
- Private evidence files.
- Credentials, logs, live access, outbound email, or provider data access.
