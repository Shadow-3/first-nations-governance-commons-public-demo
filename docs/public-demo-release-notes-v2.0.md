# Public Demo Release Notes v2.0

Date: 2026-06-05  
Package: First Nations Governance Commons Public Demo  
Mode: fake/sample data only

## Summary

v2.0 adds the Sandbox Launcher, a practical bridge from the static public demo and v1.9 Assurance Room into a controlled fake-data Decidim sandbox build.

## Added

- `sandbox-launcher.html`.
- Four launch pathway comparison: provider-assisted, Barayamal developer self-build, partner-hosted, and static-only hold.
- Current environment snapshot based on local checks.
- 14 launch readiness checks.
- 10 required Decidim setup outputs.
- Generated downloadable launch brief.
- Sandbox launch playbook.
- Decidim environment options note.
- Launch checklist CSV.
- Pathway comparison CSV.
- Decidim sandbox output map CSV.
- Sandbox launch readiness JSON.

## Improved

- The package now distinguishes "safe to hand over" from "ready to build".
- The current local environment gap is visible instead of hidden.
- The recommended next build path is explicit: provider-assisted sandbox unless the self-build environment is prepared first.
- Decidim setup outputs are clearer for quotes, delivery, QA, and handover.

## Safety Boundary

v2.0 still does not approve a real pilot or real consultation.

The next build remains fake/sample data only. Real community participation requires separate community authority, privacy, data sovereignty, cultural safety, moderation, eligibility, hosting, and support approvals.
