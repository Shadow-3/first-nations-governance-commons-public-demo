# Toastmasters First Nations Private Pilot Test Script v2.6

Release: v2.6 Private Pilot Readiness  
Mode: controlled private-pilot rehearsal script with fake/sample public evidence only

## Purpose

Use this script to rehearse whether Toastmasters First Nations can move from public coordination into a private pilot without leaking real data or confusing official Toastmasters membership with Governance Commons participation.

## Pre-Test Conditions

Do not begin until:

- The pilot authority is named.
- Privacy notice and consent text are approved.
- Minimum field list is locked.
- Role/access matrix is approved.
- Private environment is separated from the public demo and public GitHub repository.
- Official Toastmasters applications, signatures, member numbers, fees, and charter roster remain outside Governance Commons public materials.

## Test Flow

1. Open the public Organisation Onboarding sample.
2. Confirm the sample says it is not official membership infrastructure.
3. Open Private Pilot Readiness.
4. Check required gates and hold gates.
5. Run the fake or redacted private workflow rehearsal.
6. Confirm each scenario outcome.
7. Confirm no private field appears in screenshots, GitHub, public CSV/JSON, or public Decidim exports.
8. Run withdrawal and deletion rehearsal.
9. Run incident response rehearsal.
10. Hold a go/no-go meeting using the generated readiness brief.

## Scenario Acceptance

The private pilot can proceed only if:

- Demo-first contacts are not counted as charter-ready members.
- Founding-member candidates are routed to official paperwork outside the public demo.
- Access needs are visible only to the named support role.
- Organisation supporters cannot access member data or official records.
- Payment and sponsorship reconciliation stay outside Governance Commons.
- Official paperwork is logged as externally managed, not stored in the public package.
- Evidence uses fake, aggregate, or redacted records only.
- Closeout, retention, export, and deletion owners are named.

## Stop Conditions

Stop immediately if:

- A real private field is exposed in public evidence.
- Public demo data is used as the official Toastmasters roster.
- A supporter organisation is given member-data access.
- An incident has no owner or response clock.
- The pilot advances because the page looks ready rather than because gates are evidenced.

## Output

Create a short decision note:

- Proceed to private pilot.
- Hold and resolve named blockers.
- No-go for now.

The decision note must not include private records.
