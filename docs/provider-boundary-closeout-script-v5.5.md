# Provider Boundary Closeout Script v5.5

Use this script when a provider asks for real records, official records, outbound email, credentials, logs, or live access.

Approved refusal line:

"This review is fake/sample-only. We cannot provide real participant, organisation, Toastmasters, credential, log, or evidence records."

Safe alternatives:

- Synthetic sample schema.
- Fake guest/member-boundary records.
- Redacted synthetic logs.
- Private proof placeholder.
- Least-privilege temporary account only after owner approval.

Any provider request that repeats a real-record ask remains a stop trigger.
