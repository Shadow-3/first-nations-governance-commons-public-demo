# Public Demo Release Notes v5.0

The v5.0 upgrade adds a Private Decision Execution Control Pack on top of the v4.9 owner meeting pack.

## What Changed

Added an execution control page, five docs, ten CSV registers, and a machine-readable JSON package with 186 execution records.

## Why It Matters

The package now connects decisions to accepted evidence, owner actions, provider/build activation, acceptance tests, rollback triggers, custody, public communications, and next review triggers.

## Recommended Decision

Use v5.0 for controlled fake/sample execution only. Keep real participant intake, public onboarding, official Toastmasters workflows, provider access to real records, and live Decidim access blocked unless every critical owner ballot and evidence gate passes privately.
