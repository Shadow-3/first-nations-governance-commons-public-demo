# Public Demo Release Notes v4.8

The v4.8 upgrade adds a Pilot Decision and Build Handoff Room on top of the v4.7 rehearsal control room.

## What Changed

- Added `pilot-decision-build-handoff.html` as the current decision and technical handoff room.
- Added owner votes, blocker closure, Decidim technical handoff, provider/self-build comparison, launch lane policy, communications controls, implementation tickets, and evidence-to-decision mapping.
- Added 146 handoff records across nine decision/build datasets.

## Recommended Decision

Use v4.8 for private decision and build handoff only. Keep real participant intake, public onboarding, official Toastmasters workflows, and live Decidim access blocked until every critical owner vote passes privately.

## Public Boundary

This release contains public fake/sample data only. It does not approve a real pilot, real consultation, official Toastmasters process, live Decidim access, real participant onboarding, or provider access to private records.
