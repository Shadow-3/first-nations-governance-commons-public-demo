# Go/No-Go Scorecard

Status: v0.1 decision tool  
Project: First Nations Governance Commons  
Demo mode: public demo with fake/sample data only

## Purpose

Use this scorecard after advisor review to decide whether to move to a Decidim sandbox, revise the public demo, or pause.

## Scoring

Score each area from 1-5.

| Score | Meaning |

| 2 | Weak / major fixes needed |
| 3 | Adequate for demo, not pilot |
| 4 | Strong enough for sandbox |
| 5 | Strong and ready for next stage |

## Automatic No-Go Conditions

Do not proceed to a real pilot if any of these are unresolved:

- Real authorising body not confirmed.
- Eligibility/voting rules unclear.
- Cultural safety rules not approved.
- Data ownership/hosting/export/deletion unclear.
- Privacy notice missing.
- Offline participation process untested.
- Local admins not identified.
- Partner roles unclear.
- Any real sensitive data appears in public demo or sandbox.

## Decision

Choose one:

- **Go to Decidim sandbox**: public demo is strong enough for fake-data sandbox configuration.
- **Revise public demo**: fix visible/process issues before sandbox.
- **Pause for governance review**: authority, data, cultural safety, or legitimacy needs more work.
- **Prepare controlled pilot**: only if sandbox and governance gates have passed.
