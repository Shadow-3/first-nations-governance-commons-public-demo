## Public Safety Rule

The launch page is still a public fake/sample template. Real private pilot data must stay in a separate approved private environment. The public package must never include real contacts, member records, access needs, incidents, meeting links, official Toastmasters forms, member numbers, payment details, signatures, authority records, or private decision records.
