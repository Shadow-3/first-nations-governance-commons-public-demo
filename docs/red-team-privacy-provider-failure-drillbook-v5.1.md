# Red-Team, Privacy, and Provider Failure Drillbook v5.1

Run these drills with fake/sample data only.

## Required Drills

- Red-team attempts to bypass authority, privacy, data lifecycle, support, official-boundary, security, access, and public wording controls.
- Provider failure drills test whether an external builder can request, infer, retain, export, delete, or mishandle real records.
- Privacy drills prove outbound email, consent, withdrawal, support, and breach routes stay blocked until accepted privately.
- Rollback drills prove pause, revoke, destroy, archive, and public hold lines can be used without delay.

## Evidence Rule

Store proof privately. Public pages may show aggregate stress status only.
