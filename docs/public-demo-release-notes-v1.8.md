# Public Demo Release Notes v1.8

Date: 2026-06-04  
Package: First Nations Governance Commons public demo  
Mode: public fake/sample data demo only

## Summary

v1.8 adds the Delivery Room, a sprint execution cockpit that turns the live GitHub milestone and issue map into a usable day-to-day delivery board.

## Added

- `delivery-room.html` with a live-issue snapshot board, status columns, issue links, local status simulation, blocker panel, day-one run order, filters, search, and downloadable sprint update.
- `docs/sprint-execution-playbook-v1.8.md`.
- `docs/github-projects-scope-fix-v1.8.md`.
- `data/sprint-execution-tracker-v1.8.csv`.
- `data/delivery-readiness-snapshot-v1.8.json`.

## Improved

- Review Pack now includes Delivery Room as a first-class sprint team entry.
- Document viewer catalog now includes v1.8 execution docs and data.
- README, package index, and manifest now describe the v1.8 execution layer.
- Public room navigation now links to Delivery Room.

## Why It Matters

v1.7 proved that the GitHub sprint exists. v1.8 makes that sprint easier to run by showing:

- What is blocked.
- What is ready.
- Which issue should be worked next.
- Where the real GitHub issue lives.
- What evidence each issue needs.
- How to keep moving without native GitHub Projects access.

## Safety Boundary

This remains fake/sample data only. No real community records, member lists, cultural material, real voting data, eligibility data, or real decision outcomes are included.
