# Closeout Session Runner v5.6

v5.6 adds a run-ready layer on top of the v5.5 owner closeout sprint pack.

Use it to rehearse the private closeout meeting with fake/sample data:

- Check owner attendance and accepted alternates.
- Timebox the private agenda.
- Simulate evidence votes and blockers.
- Test provider refusal lines.
- Test Barayamal and First Nations Toastmasters sample boundaries.
- Approve public-safe lines without exposing private evidence.
- Export a private minute shell and owner action queue.

The runner cannot approve launch. It prepares a private minute for real owner review.
