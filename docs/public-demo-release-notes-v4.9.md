# Public Demo Release Notes v4.9

The v4.9 upgrade adds a Private Owner Decision Meeting Pack on top of the v4.8 Pilot Decision and Build Handoff Room.

## What Changed

- Added `private-owner-decision-meeting.html` as the current private owner decision meeting pack.
- Added owner ballots, quorum checks, blocker resolutions, provider RFQ questions, Decidim build ticket import, minutes, public status lines, no-go/archive routes, and owner actions.
- Added 170 meeting records across ten decision-meeting datasets.

## Recommended Decision

Use v4.9 to run the private owner decision meeting only. Keep real participant intake, public onboarding, official Toastmasters workflows, and live Decidim access blocked unless every critical owner ballot passes privately.

## Public Boundary

This release contains public fake/sample data only. Private minutes, owner reasons, evidence, participant details, official Toastmasters records, credentials, support cases, cultural material, and business-sensitive information must stay out of the public package.
