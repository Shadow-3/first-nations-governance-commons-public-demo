# Sprint Acceptance And Evidence Plan v1.5

## Evidence To Capture

- Sandbox homepage screenshot.
- Organisation settings screenshot.
- Assembly page screenshot.
- Participatory process page screenshot.
- Proposals component screenshot.
- Meetings component screenshot.
- Survey component screenshot.
- Accountability/progress screenshot.
- Fake-data labels screenshot.
- Admin role/handover notes.
- Acceptance matrix result.
- Known limitations list.

## Acceptance Checks

Use:

- `data/sandbox-sprint-issues-v1.5.csv`
- `data/sprint-evidence-log-v1.5.csv`
- `data/sandbox-acceptance-matrix-v1.3.csv`

## No-Go Evidence

Stop and remove exposure if any real community data, member data, cultural material, voting data, or sensitive personal information appears.

## Handover Evidence

Before handover, record:

- Admin access holder.
- Backup/restore note.
- Update/support owner.
- Known risks.
- Unresolved decisions.
