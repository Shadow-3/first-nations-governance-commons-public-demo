# Toastmasters First Nations Private Pilot Launch Plan v2.9

Release: v2.9 Private Pilot Launch Control  
Initial developer/custodian: Barayamal / Dean Foley  
Sample initiative: Toastmasters First Nations  
Mode: controlled public sample and private-pilot launch template only

## Positioning

Toastmasters First Nations can be used as a controlled sample to test whether First Nations organisations and initiatives can join the Governance Commons model.

The public demo is not official Toastmasters membership infrastructure. It does not hold official applications, member numbers, signatures, payment details, charter acceptance, meeting links, or private member records.

## Launch Objective

Run a controlled first 30 days that answers whether interested people understand the pathway, whether founder/supporter/sponsor/access-support routes can be handled safely, whether official Toastmasters handoff stays outside the public package, and whether Barayamal can decide proceed, hold, or no-go from evidence.

## Controlled Launch Tracks

| Track | Use | Public Rule |
| --- | --- | --- |
| Public interest | Explains the sample and invites controlled EOI through private route | Public page has no real contact form in this static package |
| Demo-first | Lets people attend an information/demo session before founder commitment | Meeting link is private only |
| Founder candidate | Tracks people who may count toward a future official path | Aggregate count only |
| Supporter or mentor | Lets people support without becoming founder candidates | No member data access by default |
| Sponsor enquiry | Keeps sponsorship separate from member records and payments | No payment details in Governance Commons public files |
| Access support | Provides support without exposing needs publicly | Support details stay private |
| Withdrawal or correction | Lets people leave or correct information | Public summary only counts requests if safe |
| Official handoff | Moves official process outside public Governance Commons package | No official forms, signatures, numbers, or payment records public |

## Launch Governance

- Barayamal / Dean Foley remains initial developer-custodian for the public package.
- A named pilot authority approves any move from template to real controlled intake.
- A privacy/data reviewer approves collection, access, retention, deletion, and public evidence rules.
- A Toastmasters launch lead owns official-process boundaries.
- A safety/moderation reviewer owns pause wording and incident closeout.

## No-Go Triggers

- Real contact, membership, payment, access-need, official paperwork, incident, meeting-link, or authority data appears publicly.
- Supporters or demo-first attendees are counted as charter-ready founders.
- Official handoff is mixed into the public demo.
- Withdrawal, correction, access support, or incident processes have no owner.
- Closeout happens without deletion, retention, and learning evidence.

## Recommended Next Step

Use the v2.9 launch page as a public walkthrough and use the private pilot package only inside an approved private environment if Barayamal proceeds with the controlled sample.
