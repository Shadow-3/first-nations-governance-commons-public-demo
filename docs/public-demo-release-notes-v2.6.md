# Public Demo Release Notes v2.6

## What Changed

v2.6 adds Private Pilot Readiness, a control layer for deciding whether the Toastmasters First Nations controlled sample can move from public coordination into a private, approval-based pilot.

## Why It Is Better

v2.5 made public organisation onboarding safe. v2.6 makes the next transition safer by adding:

- Private pilot readiness gates.
- Private workflow test scenarios.
- Role and access rules.
- Data transfer boundaries.
- Incident and closeout controls.
- A generated readiness brief.

## Safety Boundary

This is still public fake/sample evidence only. It does not collect, store, or publish real member records, contact details, access needs, payments, signatures, member numbers, official Toastmasters applications, private organisation contacts, or real authority data.
