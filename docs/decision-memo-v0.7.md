Use the public demo ZIP only.

- Momentum may stall before Decidim proof.

### Option 2: Advisor Review Then Decidim Sandbox

Run advisor review, update the demo, then build a fake-data Decidim sandbox.

- The aim is a credible funder/builder conversation.

Not recommended.

Reason:
