# Owner Vote and Blocker Resolution Guide v5.6

Voting rules:

- Owners vote only on their own gates.
- Alternates must be accepted before voting.
- Dean Foley / Barayamal can advise and review but cannot replace owner approvals.
- A held vote becomes an owner action, not a pass.
- A blocked vote keeps repeat P0 testing on hold.

Blocker rules:

- Any real participant, organisation, business-sensitive, official Toastmasters, provider, credential, log, email, or live-access record request is a stop trigger.
- Public copy must say preparation only.
- Private evidence remains outside the public package.
