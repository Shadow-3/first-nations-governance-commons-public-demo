# Private Rehearsal Closeout Playbook v4.0

## Review finding

v3.9 created a decision-ready scorecard, but the project still needed a controlled execution and closeout board for the private fake-data rehearsal itself.

## Operating rule

Use the Private Rehearsal Closeout Board after v3.9. It turns the scorecard into a controlled private fake-data execution plan: run scripts, record private evidence status, close blockers, prepare a public-safe update, and choose hold, second rehearsal, no-go, or tightly controlled pilot readiness.

## How to use this board

1. Create a private fake-data Decidim sandbox.
2. Create only fake/sample accounts from the run register.
3. Run each required Barayamal, First Nations Toastmasters, and cross-track script.
4. Store private evidence only in the approved private evidence area.
5. Mark every gate Pass, Hold, or Blocked.
6. Choose hold, second rehearsal, no-go, archive, or controlled pilot readiness.
7. Publish only the approved public-safe update.
