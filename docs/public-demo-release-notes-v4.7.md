# Public Demo Release Notes v4.7

The v4.7 upgrade adds a Pilot Rehearsal Control Room on top of the v4.6 Controlled Pilot Execution Pack.

## What Changed

- Added `pilot-rehearsal-control-room.html` as the current rehearsal control room.
- Added a time-boxed rehearsal schedule, owner sign-off board, facilitation scripts, issue triage, change requests, sample-safe roster, acceptance scorecard, decision options, and handoff actions.
- Added 132 control records across nine rehearsal-control datasets.

## Recommended Decision

Use v4.7 for private fake/sample rehearsal control only. Keep real participant intake, official Toastmasters workflows, and live Decidim access blocked until all critical owner signoffs pass privately.

## Public Boundary

This release contains public fake/sample data only. It does not approve a real pilot, real consultation, official Toastmasters process, live Decidim access, or real participant onboarding.
