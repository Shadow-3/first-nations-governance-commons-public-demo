# Public Demo Release Notes v1.3

Date: 2026-06-04  
Initial developer/custodian: Barayamal / Dean Foley  
Mode: public demo with fake/sample data only

## What Improved

- Added `build-room.html` as a builder-facing Decidim sandbox implementation cockpit.
- Added Decidim sandbox blueprint, implementation sprint plan, and admin handover/training plan.
- Added component map, acceptance matrix, implementation sprint CSV, and import manifest.
- Added filterable build streams for spaces, components, data, safety, and handover.
- Added downloadable builder brief generated in the browser.
- Added Builder Room links from the demo, Action Center, Decision Room, Review Pack, and document viewer.
- Added v1.3 files to the manifest and document viewer catalog.
- Improved mobile table rendering in the document viewer so CSV/Markdown tables collapse into labelled rows instead of requiring sideways reading.

## What Did Not Change

- The demo remains dependency-free static HTML/CSS/JS.
- All data remains fake/sample data.
- The package is still not a live Decidim instance.
- The Builder Room is an implementation guide, not a provider contract by itself.

## Release Rule

Open-source the infrastructure. Do not open-source community authority.
