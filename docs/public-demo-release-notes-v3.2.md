# Public Demo Release Notes v3.2

## What changed

- Added Activation Drill & Decision Record.
- Added decision sequence, signoff matrix, tabletop scenarios, evidence packet, fake-data rehearsal tasks, and first-72-hour monitoring checks.
- Added a browser-generated activation decision record.
- Added v3.2 playbook, Toastmasters activation drill plan, decision record template, incident/breach tabletop script, CSV files, and machine-readable JSON package.

## Review fix

v3.1 defined what must be approved. v3.2 now tests whether the approval can be run, documented, paused, and monitored before real intake opens.

## Safety rule

This public v3.2 activation drill package is fake/sample only. It must not collect or expose real names, emails, access needs, incidents, official Toastmasters records, member numbers, payments, signatures, authority records, access logs, decision records, or private pilot evidence.
