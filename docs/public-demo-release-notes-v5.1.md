# Public Demo Release Notes v5.1

## Upgrade

v5.1 adds the Private Evidence Stress Test Lab.

The v5.0 package controlled execution after the private owner meeting. The v5.1 package tests whether those controls survive failure conditions before any live pilot, real intake, provider real-record access, outbound email, or live Decidim access.

## Safety

This remains a public fake/sample data demo. Do not add real participant data, official Toastmasters records, business-sensitive information, private minutes, evidence files, credentials, support cases, cultural material, or live access details.
