# Public Demo Release Notes v1.2

Date: 2026-06-04  
Initial developer/custodian: Barayamal / Dean Foley  
Mode: public demo with fake/sample data only

## What Improved

- Added `decision-room.html` as an interactive sandbox readiness and decision tool.
- Added a weighted readiness assessment for advisor review, no-go status, provider quotes, provider score, hosting/support clarity, handover clarity, and fake-data boundary.
- Added automatic recommendation labels: proceed, hold, or no-go.
- Added downloadable decision summary Markdown generated in the browser.
- Added sandbox readiness gates, decision brief template, and decision meeting agenda.
- Added readiness checklist and sample scoring CSV files.
- Added Decision Room links from the demo, Action Center, Review Pack, and document viewer.
- Added v1.2 files to the manifest and document viewer catalog.

## What Did Not Change

- The demo remains dependency-free static HTML/CSS/JS.
- All data remains fake/sample data.
- The package is still not a live Decidim instance.
- The Decision Room can only support a fake-data sandbox decision.

## Release Rule

Open-source the infrastructure. Do not open-source community authority.
