# Public Demo Release Notes v2.5

Release date: 2026-06-05

## Summary

v2.5 adds Organisation Onboarding, a controlled sample layer for testing whether First Nations organisations or initiatives can join the Governance Commons model.

The controlled sample is Toastmasters First Nations.

## Why It Matters

The previous package could manage provider selection and delivery, but it did not yet answer how a First Nations organisation or initiative joins. v2.5 adds that missing onboarding pathway while keeping real intake, private records, official applications, payment, and member data outside the public demo.

## Public Demo Boundary

All Toastmasters First Nations records in this package are sample/test records. Real intake remains on the Barayamal page and official Toastmasters paperwork.
