## Safety Rule

The public demo remains fake/sample only. Real First Nations Toastmasters private pilot data must use a separate approved private environment with named operators, privacy controls, official handoff boundaries, incident response, retention, deletion, and closeout evidence.

## Recommended Use

Use v2.8 to run a private pilot preparation sprint. Do not open real intake until the required implementation actions, required environment controls, never-public field rules, and access register are approved.
