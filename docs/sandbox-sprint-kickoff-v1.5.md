# Sandbox Sprint Kickoff v1.5

Date: 2026-06-04  
Initial developer/custodian: Barayamal / Dean Foley  
Mode: fake-data Decidim sandbox sprint only

## Purpose

This kickoff pack turns Contributor Room and Builder Room into the first actionable fake-data Decidim sandbox sprint.

## Sprint Goal

Create a running fake-data Decidim sandbox that can be reviewed publicly without real community records, member data, cultural material, real voting data, or sensitive personal information.

## Sprint Boundary

Allowed:

- Decidim installation and configuration.
- Fake/sample data import or manual entry.
- Public warning labels.
- Admin handover notes.
- QA screenshots and acceptance evidence.

Blocked:

- Real community records.
- Member lists.
- Cultural material.
- Real voting or eligibility data.
- Claims of real mandate.

## Start Conditions

- v1.5 public package is live.
- Decision Room does not show unresolved no-go conditions.
- Contributor Room repo rules are accepted.
- Builder Room implementation map is accepted.
- Initial developer/custodian is confirmed.

## Sprint Lanes

- Setup.
- Decidim configuration.
- Safety and QA.
- Handover.

## Done Definition

The sprint is done when:

- Sandbox loads.
- Organisation, assembly, process, and components exist.
- Fake/sample data is visible.
- Public fake-data labels are visible.
- Acceptance matrix is passed or known limitations are logged.
- Admin handover notes exist.
- No real data was imported.
