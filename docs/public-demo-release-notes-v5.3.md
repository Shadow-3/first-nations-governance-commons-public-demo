# Public Demo Release Notes v5.3

v5.3 adds the Private Run Results Decision Board.

The v5.2 package prepared the first private fake/sample stress-lab run. v5.3 adds the public-safe post-run decision layer:

- 11 P0 outcome records.
- 16 evidence acceptance records.
- 18 issue records.
- 28 controlled sample results.
- 10 provider-boundary results.
- 12 owner signoffs.
- 8 final decision records.
- 10 public summary lines.
- 16 next actions.
- 129 total decision records.

This release makes the gate stricter: limited private pilot remains blocked until every P0 outcome passes, every P0 evidence item is accepted, every P0 issue is closed, and every required owner signs off privately.
