# Security Policy

## Supported Scope

This repository supports a public static demo and fake-data Decidim sandbox planning materials.

It does not support real community records, member data, cultural material, real voting data, or sensitive personal information.

## Reporting A Concern

Open a GitHub issue only if the concern can be safely described without disclosing private, sensitive, or culturally restricted information.

For sensitive issues, contact the maintainer through a private channel outside this public repository.

Initial maintainer: Barayamal / Dean Foley.

## Immediate Removal Rule

If real personal, community, cultural, or sensitive material is added by mistake:

1. Stop work on the affected issue or pull request.
2. Remove public exposure as quickly as possible.
3. Record only a minimal public note that unsafe material was removed.
4. Do not repeat the sensitive content in issue comments, commits, screenshots, or reports.
